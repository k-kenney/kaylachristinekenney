Welcome !

Let's get you started on a new project with Stimulus, a JavaScript framework by Hotwire. You'll find all of your controllers inside the javascript/controllers directory.

  • Postcss

    A tool for transforming CSS with JavaScript. It is configured with Autoprefixer, a plugin to ensure cross browser compatibility, and Purgecss, a plugin that removes unused CSS from your project. Postcss' configuration can be found in postcss.config.js

  • Sass

    A powerful CSS preprocessor that extends the capabilities of traditional CSS with features like variables, nesting, mixins, and more, making it easier to write and maintain complex stylesheets for web development projects.

  • Tailwindcss

    A utility-first CSS framework packed with classes like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup. Tailwindcss' configuration can be fond in tailwind.config.js

  • Vite

    A lightning-fast JavaScript bundler and development server that leverages ES modules to achieve near-instantaneous development server startup and hot module replacement. Vite's configuration can be found in vite.config.js

  • Deployment

    Automatically deploy your website on Github with the help of the deploy.yml github action, all the instructions you need are available in the README.md. Consider using Netlify if your deployment needs to not comply with Github's limitations.
